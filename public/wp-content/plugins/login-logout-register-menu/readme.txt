== Description ==

Login Logout Register Menu plugin allows you to add login, logout, register and profile links in the navigation menu.

The plugin provides you various settings in the admin area which you can use to customize it.

You can also use any of the following plugin shortcodes to display Login, Logout, Register and Profile links in the page content.

[llrmlogin]      -- Displays login link
[llrmregister]   -- Displays register link
[llrmloginlogout]-- Displays login or logout link depending on whether the user is logged in or not
[llrmprofile]    -- Displays profile link
[llrmlogout]     -- Displays logout link

== Installation ==

* Install the plugin from the 'Plugins' section in your dashboard (Go to `Plugins -> Add New -> Search` and search for Login Logout Register Menu).
* Alternatively, you can [download](http://downloads.wordpress.org/plugin/login-logout-register-menu.zip "Download Login Logout Register Menu") the plugin from the repository. Unzip it and upload it to the plugins folder of your WordPress installation (`wp-content/plugins/` directory of your WordPress installation).
* Activate it through the 'Plugins' section.
* Use the `Settings -> Login Logout Register Menu` to configure the plugin

== Frequently Asked Questions ==

= Why do the links not display on mobile site? =

Please make sure you have not selected the plugin option "Hide Links on Mobile" in the plugin settings from Settings -> Login Logout Register Menu
